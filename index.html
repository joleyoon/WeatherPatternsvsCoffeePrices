Overview

This project examines how weather patterns across coffee-growing regions align with market-wide coffee price movements. The site now includes the notebook-style visuals and geospatial analysis directly in the page, including a kriging surface for the latest precipitation snapshot.

Findings

The aggregate weather-price relationship remains modest, but the dataset still exposes clear market regimes, exchange-rate context, and regional weather extremes. These summary tables anchor the rest of the dashboard before moving into the richer visual analysis.

Geospatial

Because the coffee commodity price is market-wide and repeated across locations on each date, the geographic analysis is framed around regional weather regimes rather than claiming a unique local price by coordinate. The site shows regional point maps, regime shifts, coordinate-based model comparison, and a kriging interpolation surface.

Forecast

The forecast section uses a simple baseline model: it extends the recent 180-day price trend and wraps the projection with a residual-based uncertainty band. It is useful for presentation and directional storytelling, not for trading decisions.

Next 30 Days

Projected closing-price path if the recent trend persists without a major structural shift.

Method

This page is powered directly from the CSV and a generated JavaScript data bundle. That keeps the project easy to host as plain HTML while still showing statistical visuals and geospatial interpolation results.
